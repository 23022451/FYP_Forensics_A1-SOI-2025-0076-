This folder contains test images for the Media Analyzer tool.

To add test images:
1. Place normal JPEG/PNG files (screenshots, photos)
2. Add files with hidden content (steganography)
3. Add files with appended data (ZIP, EXE)

The Media Analyzer will detect:
- LSB steganography
- Embedded files (ZIP, RAR, EXE)
- Hidden metadata
- Executable code
- Encrypted containers
- OCR text extraction
- Video frame analysis
- PDF content analysis
- EXIF data

Test files needed:
- test_1_normal_image.jpg (clean image)
- test_2_steganography.png (with hidden data)
- test_3_malicious.jpg (with embedded executable)
